# Getting Started with Hartonomous

Quick guides to get you running in minutes.

---

## Prerequisites

Before you begin, ensure you have:
- **Docker** 24.0+ with Docker Compose
- **8GB RAM** minimum (16GB recommended)
- **10GB disk space** for initial setup
- **Git** (to clone repository)

## What You'll Learn

### Quick Start
- Start all services with one command
- Verify health endpoints
- Ingest your first document
- Run your first semantic query
- View results

### Installation
- Docker Compose setup
- Environment variable configuration
- Service health checks
- Troubleshooting connection issues

### First Ingestion
- How text becomes atoms
- Content addressing (SHA-256)
- Spatial positioning (semantic neighbors)
- Composition hierarchy
- Reference counting

### First Query
- Spatial queries (K-nearest neighbors)
- Semantic distance calculations
- Result ranking
- Provenance traversal (Neo4j)
- Relation weights
