## Technology Stack

### Core Technologies
- **.NET 10** - Latest .NET framework
- **C# 13** - Latest language features
- **EF Core 10** - ORM with PostgreSQL provider
- **PostgreSQL 16+** - Primary database
- **PostGIS** - Spatial database extension
- **PL/Python** - GPU-accelerated database functions

### Infrastructure
- **Redis** - Distributed caching
- **RabbitMQ** - Message broker (optional)
- **Azure Services** - Cloud infrastructure

### Frontend
- **Blazor** - Web UI (Server + WebAssembly)
- **.NET MAUI** - Cross-platform apps
- **SignalR** - Real-time communication

### DevOps & Observability
- **.NET Aspire** - Local orchestration
- **OpenTelemetry** - Observability
- **Docker** - Containerization
- **GitHub Actions** - CI/CD

### Testing (To be added)
- **xUnit** - Unit testing
- **FluentAssertions** - Test assertions
- **Moq** - Mocking framework
- **Testcontainers** - Integration testing
